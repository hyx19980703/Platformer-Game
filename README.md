# 2D平台跳跃游戏

![游戏截图](screenshots/gameplay.gif) <!-- 如果有截图或GIF -->

## 🎮 项目概述
一个基础的2D平台跳跃游戏，角色拥有扔炸弹的能力，需要利用该能力完成谜题通关。  
**核心玩法**：  
- 控制角色跳跃&炸弹的能力达成谜题通关
