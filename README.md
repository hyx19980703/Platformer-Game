# 2D平台跳跃游戏

![游戏截图](screenshots/gameplay.gif) <!-- 如果有截图或GIF -->

## 🎮 项目概述
一个基础的2D平台跳跃游戏，包含角色移动、收集物品、敌人交互和关卡设计。  
**核心玩法**：  
- 控制角色跳跃躲避陷阱  
- 收集金币解锁新区域  
- 击败敌人完成关卡
